Represents a po file entry.

**Examples**:

>>> entry = POEntry(msgid='Welcome', msgstr='Bienvenue')
>>> entry.occurrences = [('welcome.py', 12), ('anotherfile.py', 34)]
>>> print(entry)

msgid "Welcome"
msgstr "Bienvenue"
